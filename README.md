# Meesam Fragrance - Perfume Portfolio Website

A professional e-commerce style website showcasing a collection of perfumes.

## Usage

Open `index.html` in a web browser to view the website.

## Features

- **Logo**: "MF" circular logo with "Meesam Fragrance" branding
- **Modern Design**: Gradient backgrounds, smooth animations, and professional color scheme
- Hero section with banner image and gradient text
- Perfume gallery with hover effects, descriptions, prices, and "Add to Cart" buttons (5 unique fragrances)
- Modal popup for detailed perfume views with enhanced styling
- Shopping cart counter in header with animations
- About and contact sections with newsletter signup
- Customer testimonials with hover effects
- Responsive design with mobile-friendly layout
- Smooth transitions and animations throughout

## E-commerce Features

- Product pricing with individual item costs
- Add to cart functionality with item count and total price display
- Shopping cart counter and total in header with real-time updates
- Professional layout suitable for online sales

## Troubleshooting

- Ensure all image files are in the `images/` folder.
- If images don't load, check file paths and extensions.
- For best viewing, use a modern web browser.

## Customization

- Edit `style.css` to change colors and layout.
- Modify `script.js` for additional interactivity.
- Update `index.html` to add more perfumes or sections.
